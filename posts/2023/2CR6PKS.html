重装 windows 系统

发表于 2023-08-07 分类于 System ， Windows 本文字数： 1.6k 阅读时长 ≈ 1 分钟

本文介绍如何彻底重装 windows 系统，按照本教程一步一步进行安装，重装系统非常简单。

重装相对于重置可以让系统更纯净。有时候电脑被病毒感染后，重置往往不能解决问题，此时就需要进行重装。

若只需要重置电脑，win10 及以上的版本在设置有重置功能，可以就此跳过本文。

简介

通过本文的方法安装 windows 有以下几个优势：

  1. 安装方式简单，不影响 U 盘的正常使用
  2. 安装的系统为官方iso，无任何其它捆绑软件
  3. 激活后为正版系统，支持升级，放心使用

安装盘制作

从 Download . Ventoy 处下载 ventoy-1.0.94-windows.zip，然后根据 Get start . Ventoy 教程制作启动盘。

简要步骤如下：

  1. 插入 U 盘

  2. 解压 ventoy 安装包后，执行文件 Ventoy2Disk_X64.exe

  3. 选择目标 U 盘，然后单击【安装】，便完成启动盘的制作了

    当出现如下提示时，表示安装成功了：

windows 系统 iso 下载

使用 microsoft 提供的工具 下载 Windows 10 (microsoft.com) 下载安装镜像(.iso)文件。

更详细的教程可以在网络上找一下，应该不难

下载完成后，将下载的 .iso 文件复制到刚刚制作的 U 盘中即可。

简要步骤如下：

  1. 下载 microsoft 提供的下载工具

  2. 打开下载好的 MediaCreationTool22H2.exe

    点击【下一步】，然后选择保存 iso 文件的位置即可。

开始安装 Windows

备份 C 盘内容

先备份目标机器上 C 盘中的文件，包括桌面上的内容。在系统重装时，会清理原 C 盘的内容，有时候需要格式化 C 盘，因此需要提前备份，避免数据丢失。

进入 BIOS 设置 U 盘启动

通过 BIOS 快捷键，进入 BIOS 中，设置

安装系统

当启动 U 盘后，会进入 ventoy 的系统安装界面，按以下步骤逐步操作即可：

  1. 选择需要安装的 windows 版本

  2. 选择 Boot in normal mode

  3. 安装界面启动后，一直点下一步

    当到选择操作系统时，选择【专业版】

一键激活系统

当系统安装完成，下载激活脚本MAS_AIO，双击打开即可激活。

或者按该教程激活 windows: windows 8.1/10/11 系统激活
