重置 winsock 解决电脑网络问题

发表于 2021-09-26 更新于 2024-10-22 分类于 System ， Windows 本文字数： 787 阅读时长 ≈ 1 分钟

有人反馈电脑无法上网，检查后发现以下情况：

  • 电脑的网络图标不断显示正在获取网络和获取失败
  • 把电脑连接其它正常的网线，依旧无法连接网络
  • 用其它电脑连接当前网线，能正常上网
  • 该电脑在家可以上网

基于以上情况，说明网卡硬件正常，问题集中在操作系统的软件层面。

鉴于电脑网络的复杂性，直接通过 cmd 运行Netsh Winsock reset 解决了问题。

什么是Netsh Winsock reset？

Netsh winsock reset是一个命令提示程序，用于将 winsock 目录重置为默认设置或清除状态。

如有时候上不了网或者网络出现问题经常用到它，简单地理解就是：重置程序通过操作系统链接网络的入口点。虽然使用此命令可以恢复网络连接，也应谨慎使用，因为可能需要重新安装LSPLSP: layered service privider 分层服务提供商。LSP是TCP/IP等协议的接口。

Netsh Winsock解决了哪些问题？

  • 解决删除广告软件，间谍软件，病毒，蠕虫，木马等后产生的网络问题。

  • 安装广告软件，间谍软件，VPN或防火墙后无法联网。

  • 无法访问任何网页或只能访问某些网页。

  • 出现与网络相关问题的弹出错误窗口。

  • 由于注册表错误，没有网络连接。

  • 发生DNS查找问题。

  • 无法续订网络适配器的IP地址或其他一些DHCP错误。

  • 没有连接消息的网络连接问题。

如何在Windows 10中执行 winsock reset？

  1. windows键+R或者windows键+x选择（管理员）命令提示符

  2. 输入 cmd

  3. 在命令提示框中键入 netsh winsock reset，然后回车

  4. 完成重置后，您将收到消息“Winsock重置成功完成”。重新启动电脑完成Windows 10 中的重置 winsock

同样，要重置TCP / IP，请在 cmd 中使用命令 netsh int tcp reset。

若是 win11，可以直接在【网络和 Internet/高级网络设置/网络重置】中进行重置
